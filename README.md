<h1 align="center">INSTALLATION</h1>

Download the 'install.sh' to your /Downloads folder

open terminal 
  `cd /Downloads`
  `chmod +x install.sh`
  `./install.sh`
  ...DONE

The installation will create everything you need for programs and directories, all you need to do is bring your own GIF and add it to the newly created path $HOME/Pictures/Wallpapers/GIFs

Once your gif is ready you can CTRL+SUPER+ALT+R to restart caelestia or you can use terminal:
  `hyperctl reload` or
  `reboot`
